Hoje o tema é fontes e negrito

Negrito:

A tag <b> deixa paragrafos em negrito, a diferença, que essa tag é não semântica, uma tag de forma

Uma marcação com a tag <strong>, não tem diferença com a tag <b> a primeira vista,
mas a diferença está em seu sentido, que é uma tag semântica.

Itálico

A tag <i> coloca letras em itálico, tag de forma não semântica.

A tag <em> da ênfase a palavra, é uma tag semântica.

Essas tags de formas não são recomendadas para serem usadas, pois são tags de forma
o HTML trabalha com sentido, diferente do CSS, que trabalha com forma, lembre,
sempre use tags semânticas!
